Content Automation

RAG-powered daily content engine for a sports-psychology niche — Make.com orchestrates Gmail, schedulers, and a FastAPI RAG endpoint.

⟁ Code ↗ Live (soon) ▶ Demo (soon)

A four-stage RAG pipeline that turns a curated cricket / sports-psychology knowledge base into daily script ideas, reels, carousels, and captions — delivered straight to inbox via Make.com.

  • 4-node pipeline: structured brief → multi-query Pinecone retrieval → diversity-aware reranking → angle selection → full content pack
  • Daily Gmail delivery via Make.com scheduler hitting the FastAPI endpoint with topic + role + level inputs
  • gpt-4o-mini + text-embedding-3-small tuned for cost-per-piece, with debug mode exposing chunks & angles for QA
  • Deployed-ready on Render with render.yaml; Swagger UI for client testing

Jobber

AI-powered remote-job ops platform — discover, score, tailor resumes, and track applications for software / AI / data roles.

// 03 Production · Medpho

AI Voice Call Agent

Multilingual AI-powered triage caller built on LiveKit SIP + Twilio. Automated 70% of initial patient screening for a healthcare platform.

⟁ Code (private) ▶ Demo (NDA)

Built during my Software Engineer Internship at Medpho (Apr–Sep 2025): a real-time voice agent that pulls patients from Google Sheets via webhooks and calls them in Hindi or English for triage and follow-ups, grounded by a RAG layer for live patient queries about doctors and services.

  • LiveKit SIP + Twilio for low-latency telephony; GPT-4o mini + Sarvam AI for cost-effective multilingual STT/TTS
  • Smart retry logic for missed calls and edge cases — boosted successful contact rates and automated ~70% of initial screening
  • RAG over LangChain + Pinecone answers patient questions about services and doctors in real time, mid-call
  • Webhook-driven call-list ingestion from Google Sheets — non-engineers manage campaigns themselves
